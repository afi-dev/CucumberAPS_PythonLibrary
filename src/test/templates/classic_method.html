This Flask app is an example of using the CucumberAPS_Python library with the classic license verification method

For more details on the use refer to the more detailed documentation on the caps() function with the classic type argument

Check a CucumberAPS License with Classic Method

Note: If you do not have a license to test run create one from CucumberAPS

Flask Code:

                
@app.route('/classic_method',methods=['GET','POST'])
def classic_method():
	result=None
	if request.method=='POST':
		if request.form['license']:
			license=request.form['license'];result=caps(type='classic',license_key=license)
			if result==True:result='True'
			else:result='False'
		else:result='False'
	return render_template('classic_method.html',result=result)
